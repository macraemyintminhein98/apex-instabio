# InstaBio AI

InstaBio AI is an AI-powered Instagram bio generator. Input keywords, and get creative, engaging bio variations instantly.

## Features
- Keyword-based bio generation
- Multiple bio variations
- Simple, intuitive interface

## Getting Started
1. Enter your desired keywords (e.g., 'travel blogger', 'fitness enthusiast').
2. Click 'Generate Bios'.
3. View generated bios.
4. Purchase full access for unlimited generations.

## Tech Stack
- Frontend: HTML, TailwindCSS, JavaScript
- Backend: Vercel Serverless Function (placeholder API calls)
- Payment: Stripe Checkout (integration planned)

## Development Notes
This `index.html` provides a basic frontend structure for key input and output. The actual AI generation and Stripe integration would be handled by Vercel serverless functions.
